# TEST

## Getting Started
- Clone this repo
- ```npm install -g gulp```
- ```npm install```
- ```bower install```
- run ```gulp```

## Gulp Tasks
- ```gulp```
    - Start ```gulp build``` && add watch
- ```gulp build```
    - Rebuild all project files
- ```gulp clean```
    -   Remove ```/dist``` directory
- ```gulp js:build```, ```gulp style:build```, ```gulp html:build```, ```gulp fonts:build```, ```gulp images:build``` for selective build
